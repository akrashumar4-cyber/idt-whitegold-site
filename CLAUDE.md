# IDT — International Diamond Trade

## Project
B2B diamond marketplace connecting verified suppliers with global buyers.
Live at: app.internationaldiamondtrade.com (Vercel + Supabase)
Supabase Project ID: hlmdkhixdrowbnxgbuvy (Asia-Pacific)

## Design System
- Background: #06080F (primary), #0D1020 (cards)
- Gold: #C9A84C (primary), #E8D5A3 (light)
- Text: #E8E8F0 (primary), #8A8FAA (muted)
- Green: #4CAF7D (verified), Red: #E05555 (danger)
- Fonts: Cormorant Garamond (headings), DM Sans (body/UI)
- Style: Dark luxury B2B. Bloomberg Terminal meets luxury jewellery.

## Animation Stack
- Framer Motion v11 for scroll/layout animations
- Three.js for 3D diamond hero
- Lenis for smooth scroll
- GSAP for text reveals
